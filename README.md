## Generators

Seven "Generators" are included at the moment. Generators return a new world when called. Each generator can be controlled with up to three knobs/parameters. *Cursive** written generators are untested.

- **g_cube**
- **g_growing_sphere**
- **g_random**
- **g_randomlines**
- **g_shooting_star**
- **g_sphere**
- **g_snake**
